Đây là trang web của tôi

Chào mừng dến với trang web của tôi.Đây là một ví dụ về cách sử dụng thẻ img để hiển thị hình ảnh trên web


Trên đây là một hình ảnh ngẫu nhiên được tải từ trang web Google.
Bạn có thể thay đổi kích thước hình ảnh bằng cách chỉnh độ rộng và chiều cao trong thuộc tính "width" và "height"của thẻ.
Bạn cũng có thể thêm một mô tả hình ảnh bằng thuộc tính "atl".
